# EricHour's Personal Page

A modern, dark-themed personal homepage built with vanilla HTML/CSS/JS.

## Features

- 🌑 Dark theme with GitHub-inspired colors
- 📱 Fully responsive design
- ⚡ Fast loading - no frameworks needed
- 🔗 GitHub API integration for blog posts
- 🎨 Clean and modern UI

## Preview

Visit: https://zoranke.github.io

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages

## Development

```bash
# Clone the repo
git clone https://github.com/zoranke/zoranke.github.io.git

# Open in browser
open index.html
```
